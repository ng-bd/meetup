# NG-BD

Repository for the website of **[ng-bd.com](https://ng-bd.com/)**

> This repository hosts the source-code of the front-end UI.

## Project Setup
`With: yarn`
```
yarn install
```
`With: npm`
```
npm install
```

## Running ***dev*** server
```
gulp run
```
